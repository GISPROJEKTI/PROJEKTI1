## SoftGIS ohjelmisto

Kansiossa SoftGIS-master löytyy itse ohejlma ja sen asennusohje.

## Testiaineistot

Kansiossa Testiaineistot löytyy muutamia tiedostoja, joiden avulla voit kokeilla ohjelman toimintoja.
- .mif -päätteiset tiedostot voit viedä ohjelmistoon vektoriaineistoiksi.
- Kuvatiedoston voit ladata palvelimelle karttakuvaksi. Koordinaatit -tiedostossa on karttakuvan kulmien koordinaatit.
 - Jos luot itse uusia karttakuvia, valitse käytettäväksi koordinaattijärjestelmäksi WGS84, niin kuvan pitäisi olla SoftGIS ohjelmiston kartalle sopiva.

## Dokumentaatio

Kansiossa Dokumentaatio löytyy teknisempää materiaalia järjestelmästä, joka on suunnattu lähinnä ohjelmiston kehittäjille.

## SoftGIS ohjelmisto

Kansiossa SoftGIS-master löytyy itse ohejlma ja sen asennusohje.

## Testiaineistot

Kansiossa Testiaineistot löytyy muutamia materiaaleja, joiden avulla voit kokeilla ohjelman toimintoja. (Lisätään kun saadaan materiaali)

Kansiossa Dokumentaatio löytyy teknisempää materiaalia järjestelmästä, joka on suunnattu lähinnä ohjelmiston kehittäjille.
